== Description ==

This plugin extends the WP All Import and Export plugins. It will add the ability to import Visual Composer single images, which are stored as id's. It will also add the ability to import VC snippets, also stored with post_id.
You need to install WP All Import and WP All Export before installing this plugin to be able to use these features.
